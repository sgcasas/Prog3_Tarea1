## Red Neuronal

Inspirados en librerias como Numpy y PyTorch, el objetivo del proyecto es implementar una libreria llamada Tensor para recrear una red neuronal.

## Estructura del Proyecto

La clase principal es la clase Tensor, esta clase soporta operaciones básicas(sumar, resta, multiplicación), transformaciones, funciones estáticas(zeros, ones, random, arange), y además operaciones avanzadas como matmul y dot.

## Instrucciones de uso:

- Para poder usar este programa, primero es necesario convertir el código escrito en C++ en un programa que la computadora pueda ejecutar.
- Una vez que el código ha sido compilado correctamente, se obtiene un archivo main.exe. Este archivo es el programa listo para ejecutarse.
- Luego, se procede a la ejecución, que consiste en correr ese archivo ejecutable desde la terminal. Al hacerlo, el programa realiza todas las operaciones definidas, y haciendo la simulación de la red neuronal usando la clase Tensor, el modificador view, los operadores y las funciones ReLu y Sigmoid, todas estas previamente compiladas.
